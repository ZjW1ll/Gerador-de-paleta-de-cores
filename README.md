# Gerador de paleta de cores

Esse é um projeto pessoal meu para preencher portfólio, tive a ideia de fazer um gerador de paleta de cor com cores complementares e análogas

O gerador cria em hsl e faz alterações na base (hue), saturação (saturation) e na iluminação (lightness), gerando cores diferentes mas que tenham um hue próximo.
É possível alterar a paleta de cores tanto pelo botão na página, quando pelo a tecla "espaço" do teclado.
O script além de criar as cores, ele também transforma de hsl para hex, que aparece ao passar o mouse por cima da cor, ao clicar nela o código é copiado para a sua área de transferência.
